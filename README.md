# 📒 Contact Book App

## 🌟 About
A simple but powerful contact book app built with Python. 
Store your contacts safely so you never lose them again — 
even if your phone resets or storage gets wiped!

## 😤 Problem It Solves
- 📱 Phone reset = all contacts lost forever
- ⏰ Writing numbers manually = waste of time
- 😰 Losing important contacts = stressful
- 💸 No need for expensive apps — this is free!

## ✨ Features
- ➕ Add Contact
- 👀 View All Contacts
- 🔍 Search Contact
- 🗑️ Delete Contact
- 💾 Contacts saved permanently using JSON

## ▶️ How to Run
```bash
python contact_book.py
```

## 🛠️ Built With
- Python 🐍
- JSON for permanent data storage
